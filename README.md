## Relatório

#### Introdução
<p>O objetivo do trabalho realizado com o kit da Nordic Semiconductor foi para investigar suasfuncionalidades e aplicações que poderiam ser realizadas com o mesmo. O nRF52840 é uma placaque contém diversos módulos de desenvolvimento como Bluetooth 5, Bluetooth mesh, Thread,Zigbee, 802.15.4, ANT e 2.4 GHz com aplicações proprietárias no nRF52840 SoC.</p>
<p>A princípio foram trabalhados os módulos básicos de periféricos como Logger RTT,
comunicação TX/RX e conversor analógico/digital, após o desenvolvimento das atividades
primárias seria realizado o estudo da parte de Bluetooth Mesh integrado aos periféricos básicos.</p>
<p>Todas as informações presentes neste relatório foram retiradas da documentação oficial daNordic.</p>
